# مشروع تصنيف صور الحيوانات بالذكاء الاصطناعي

في هذا المشروع قمت بتدريب نموذج ذكاء اصطناعي يستطيع التعرّف على ثلاثة أنواع من الحيوانات:

- الجمل (Camels)
- الأسد (Lion)
- القرد (monkey)

استخدمت موقع **Teachable Machine** لتدريب النموذج، ثم صدّرته وشغّلته باستخدام لغة Python، وبعد ذلك اختبرته على ثلاث صور مختلفة.

## 1. تدريب النموذج باستخدام Teachable Machine

بدأت بفتح موقع [Teachable Machine](https://teachablemachine.withgoogle.com/)، ثم اخترت **Get Started** وبعد ذلك **Image Project** و**Standard Image Model**.

أنشأت ثلاث فئات داخل المشروع، وهي:

- `Camels` لصور الجمال.
- `Lion` لصور الأسود.
- `monkey` لصور القرود.

بعد ذلك أضفت مجموعة من الصور المناسبة إلى كل فئة. استخدمت صورًا متنوعة لمساعدة النموذج على التمييز بين الحيوانات بصورة أفضل.

بعد تجهيز الصور، ضغطت على **Train Model** وانتظرت حتى انتهت عملية التدريب. ثم اختبرت النموذج داخل الموقع باستخدام بعض الصور، وتأكدت من أنه يستطيع التعرّف على الفئات الثلاث.

## 2. تصدير النموذج

بعد انتهاء التدريب، صدّرت النموذج من الموقع من خلال الخطوات التالية:

1. ضغطت على **Export Model**.
2. اخترت **TensorFlow**.
3. اخترت صيغة **Keras**.
4. ضغطت على **Download my model** لتنزيل النموذج.
5. فككت ضغط الملف، ثم وضعت `keras_model.h5` و`labels.txt` داخل مجلد المشروع.

استخدمت ملف `keras_model.h5` لتحميل النموذج المدرّب داخل كود Python، بينما استخدمت `labels.txt` للاحتفاظ بأسماء الفئات.

## 3. ملفات المشروع

- `predict.py`: كود تحميل النموذج واختبار الصور.
- `keras_model.h5`: النموذج الذي تم تصديره من Teachable Machine.
- `labels.txt`: أسماء الفئات الثلاث.
- `requirements.txt`: المكتبات المطلوبة لتشغيل المشروع.
- `camel.jpg`: صورة اختبار للجمل.
- `lion.jpg`: صورة اختبار للأسد.
- `monkey.jpg`: صورة اختبار للقرد.
- `Screenshot_1.png` و`Screenshot_2.png` و`Screenshot_3.png`: صور نتائج الاختبارات.

## 4. تجهيز بيئة تشغيل المشروع

استخدمت **Python 3.13** لتشغيل المشروع، ثم أنشأت ملف `requirements.txt` الذي يحتوي على المكتبات المطلوبة.

ثبتُّ المكتبات من خلال Terminal داخل مجلد المشروع باستخدام الأمر التالي:

```bash
python -m pip install -r requirements.txt
```

وللتأكد من استخدام إصدار Python 3.13، استخدمت الأمر التالي:

```bash
py -3.13 -m pip install -r requirements.txt
```

## 5. كتابة وتشغيل كود الاختبار

كتبت ملف `predict.py` لتحميل النموذج واستقبال مسار الصورة ثم عرض التصنيف ونسبة الثقة. بعد ذلك شغّلت الملف بالأمر التالي:

```bash
python predict.py
```

بعد تشغيل البرنامج ظهر طلب إدخال مسار الصورة:

```text
Enter image path:
```

أدخلت اسم الصورة التي أردت اختبارها، مثل:

```text
camel.jpg
```

بعد إدخال اسم الصورة، قام البرنامج بعرض اسم الحيوان الذي توقعه النموذج ونسبة ثقته في النتيجة.

## 6. ما الذي نفذه الكود؟

نفذ ملف `predict.py` الخطوات التالية:

1. حمّلت النموذج من `keras_model.h5`.
2. استقبلت مسار الصورة التي أدخلتها للاختبار.
3. غيّرت حجم الصورة إلى `224 × 224` بكسل، وهو الحجم المطلوب بواسطة النموذج.
4. حوّلت الصورة إلى مصفوفة أرقام.
5. طبّعت قيم الصورة لتصبح مناسبة للنموذج.
6. أرسلت الصورة إلى النموذج للحصول على التوقعات.
7. اخترت الفئة التي حصلت على أعلى احتمال.
8. عرضت اسم الحيوان المتوقع ونسبة الثقة.

## الخلاصة

في هذا المشروع استطعت تدريب نموذج لتصنيف صور الجمل والأسد والقرد باستخدام Teachable Machine. صدّرت النموذج بصيغة Keras، ثم كتبت كود Python لتحميله واختباره. أظهرت الاختبارات الثلاثة أن النموذج تعرّف على الصور بصورة صحيحة وبنسب ثقة مرتفعة.
